# 사운드 파일 안내

이 폴더에 다음 MP3 파일들을 추가해주세요:

## 필요한 파일들:

1. **type1.mp3** - 타자 소리 1 (일반 키 입력시 사용)
2. **type2.mp3** - 타자 소리 2 (일반 키 입력시 사용)
3. **type3.mp3** - 타자 소리 3 (일반 키 입력시 사용)
4. **bell.mp3** - 벨 소리 (엔터키 입력시 재생)
5. **winding.mp3** - 와인딩 소리 (스페이스바 입력시 재생)

## 사용법:
- 일반 키보드 타이핑: type1.mp3 → type2.mp3 → type3.mp3 순서로 순회하며 재생됩니다
- Enter 키: bell.mp3가 재생되며 ⏎ 표시
- 스페이스바: winding.mp3가 재생되며 SPACE 표시
- 백스페이스: type1/type2/type3가 순회하며 재생되고 ⌫ 표시

## 사운드 파일 구하는 방법:

### 무료 사운드 사이트:
- Freesound.org
- Zapsplat.com
- Adobe Stock (무료 계정)
- YouTube Audio Library

### 검색 키워드:
- type1.mp3, type2.mp3, type3.mp3: "typewriter key", "mechanical keyboard", "vintage typewriter"
- bell.mp3: "typewriter bell", "ding sound", "notification bell"
- winding.mp3: "typewriter winding", "carriage return", "mechanical winding"

## 파일 형식:
- MP3 형식 권장
- 파일 크기: 1MB 이하 권장 (빠른 로딩)
- 길이: 1-2초 정도의 짧은 사운드

## 테스트:
사운드 파일이 없어도 프로그램은 정상 동작하며, 콘솔에서 로딩 상태를 확인할 수 있습니다.
